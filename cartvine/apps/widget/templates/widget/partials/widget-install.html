{% trans 'How do I install this widget into my shopify store?' %}

Please Note: We automatically install the cartvine-loader.js script into your sites Assets folder; so all you need do is insert it into the base liquid file


  1. Go the "Manage my Theme" in your shopify admin page (http://your_domain.myshopify.com/admin/theme).
  2. Open the "theme.liquid" file.
  3. Paste "-insert-script-here-" just before the end of the "<body>" tag

Please Note: If you have already done this for another of our widgets, then you dont need to do it again for this widget.
